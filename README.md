### Hi, I'm Demir 👋

I'm a blind developer from Serbia. I studied English Language and Literature 
in Novi Sad, and only recently made the jump into programming — later than 
most, but for a reason that matters a lot to me.

**Why I code**

My son turns two this November, and my wife writes children's songs. 
Watching them, I kept coming back to the same question: so many of the apps 
people rely on every day — the ones that are genuinely useful — simply 
aren't usable if you're blind. Not because it's hard to fix, but because 
accessibility is rarely the starting point.

I want to change that, one project at a time: either by building accessible 
alternatives to popular tools, or by rethinking how existing tools approach 
accessibility from the ground up.

**What I'm working on**

- 🎬 [Ultra Creative Suite](https://github.com/demirajvazi10-max/Ultra-Creative-suite) 
  — an open-source accessibility engine for creative software (video/audio tools)
- 📷 [AI Camera Fan Edition](https://github.com/demirajvazi10-max/ai-camera-fan-edition) 
  — exploring AI-assisted camera tools
